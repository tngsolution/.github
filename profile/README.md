## üéØ Purpose

This repository serves several important functions:

1. **Consistency** - Ensures all projects follow the same standards and guidelines
2. **Onboarding** - Helps new contributors understand how to work with us
3. **Governance** - Establishes clear policies and expectations
4. **Community** - Builds a welcoming and inclusive environment
5. **Security** - Defines responsible disclosure and security practices

---

## üöÄ How It Works

GitHub automatically applies files from this `.github` repository to all organization repositories:

- When a contributor opens an issue, they see the issue templates
- Pull request templates appear when creating new PRs
- Community guidelines are referenced in various GitHub workflows
- Security policy is displayed in the security tab

Individual repositories can override these defaults by including their own versions of these files.
